Remnawave Panel & Node Manager
===============================

🚀 Автоматическая установка, управление и мониторинг Remnawave
---------------------------------------------------------------

__При поддержке Y-VPN | Создатель: @drugd | Канал: @yurichvpn__

О скрипте
---------

Один скрипт `remnawave-manager.sh` даёт полный контроль над панелью Remnawave и RemnaNode.
Выберите раздел при запуске — и управляйте всем через удобное меню с подсказками,
прогресс-барами и цветным интерфейсом.

Что умеет
---------

🖥️  __Панель Remnawave__
   - Установка с нуля (клонирование, настройка .env, Docker Compose)
   - Просмотр и редактирование переменных окружения
   - Обновление до последней версии (git pull + пересборка)
   - Просмотр логов всех сервисов в реальном времени
   - Статус контейнеров (docker-compose ps)
   - Проверка установленной версии
   - Полное удаление с очисткой данных

📡 __RemnaNode__
   - Локальная установка ноды и подключение к панели
   - Просмотр логов через journalctl
   - Статус сервиса remnanode
   - Проверка версии
   - Удаление ноды с удалением конфигурации

🎨 __Интерфейс__
   - Анимированные спиннеры и линейные прогресс-бары
   - Понятные подсказки и сообщения об ошибках
   - Обработка прерываний и сбоев с указанием строки ошибки
   - Автоматическая установка Docker и docker-compose при необходимости

Быстрый старт
-------------

```
git clone https://github.com/Pykucyka/remnasetup.git
cd remnasetup
chmod +x remnawave-manager.sh
./remnawave-manager.sh
```

Для установки локальной ноды нужны права root. Скрипт предупредит, если запущен не от root.

Меню скрипта (главное)
-----------------------

При запуске вы попадаете в главное меню:

   1. Панель Remnawave (установка, обновление, управление)
   2. RemnaNode (установка, логи, удаление)
   0. Выход

После выбора раздела открывается соответствующее меню со всеми действиями.

Структура репозитория
---------------------

```
remnasetup/
├── remnawave-manager.sh   ← главный исполняемый скрипт
└── README.md
```

После установки панели появится папка `remnawave-panel/` с docker-compose.yml и .env.

Требования
----------

- Ubuntu 20.04 / 22.04 / 24.04 (Debian 11/12 тоже подойдут)
- Доступ в интернет
- Права root (рекомендуется, особенно для ноды)
- Docker и docker-compose (установятся автоматически, если их нет)

Безопасность
------------

- Все пароли и ключи генерируются случайно
- Файл .env содержит конфиденциальные данные — не публикуйте его
- Используются только официальные репозитории Remnawave
- При удалении панели контейнеры и тома Docker полностью очищаются
